## 📲 Downloading MaheshStream App

### For Android Users

**Automatic Download (Recommended):**

1. Visit **https://maheshstream.com**
2. **Age Verification** - Check the box "I am 18+"
3. **Install Prompt** - Tap "Download App" at top of download section
4. **APK Downloads** - File downloads to your Downloads folder
5. **Install** - Open Downloads, tap APK, tap "Install"
6. **Open App** - Launch MaheshStream from your app drawer

**Manual Download:**

1. Visit **https://maheshstream.com**
2. Scroll to "Download" section
3. Click "Download APK (18.5 MB)" button
4. Right-click and select "Save Link As..."
5. Save file to Downloads folder
6. Open file and install

### For Desktop Users

1. Visit **https://maheshstream.com**
2. Click "Download APK" button
3. Right-click and select "Save Link As..."
4. Save to computer
5. Transfer APK to Android device via USB
6. Install on Android device

## ⚙️ Installation Requirements

### Minimum Requirements:
- Android 6.0 (Marshmallow) or higher
- 100 MB free storage space
- Stable internet connection
- Unknown sources enabled

### How to Enable Unknown Sources:

**Android 8.0+**:
1. Settings → Security & privacy
2. Find "Install unknown apps"
3. Select your browser/file manager
4. Enable installation permission

**Android 6.0-7.1**:
1. Settings → Security
2. Enable "Unknown sources"

## ❓ Troubleshooting

### APK Won't Download
- Check internet connection
- Disable ad blocker
- Try another browser
- Use direct download link

### App Won't Install
- Enable "Install from unknown sources"
- Check storage space
- Clear Downloads folder
- Try downloading again

### App Crashes on Launch
- Update to latest version
- Clear app data
- Reinstall app
- Check Android version compatibility

### Auto-Update Not Working
- Open app weekly to check for updates
- Check internet connection
- Ensure storage space available
